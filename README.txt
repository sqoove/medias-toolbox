Medias Toolbox allow you to sanitize and rename automatically media files during upload.

== Description ==

Medias Toolbox allow you to sanitize and rename automatically media files during upload using severals predefined patterns. Furthermore Medias Toolbox offer you the option to compress your JPG and PNG image file to optimize your website and page loading time.

== Installation ==

1. Upload "medias-toolbox" folder to the "/wp-content/plugins/" directory
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Navigate the "Medias Toolbox" menu located in your dashboard sidebar to use or configure the plugin options.

** Supported Patterns **

*	Current Date/Time + Sanitized filename
*	e.g: 201907092140-original-filename-sanitized.jpg

*	Current Timestamp + Sanitized filename
*	e.g: 1562708406-original-filename-sanitized.jpg

*	Current Date/Time + Unique Random String
*	e.g: 201907092140-95560d5755951e92fcc443295eb6685e.jpg

*	Current Timestamp + Unique Random String
*	e.g: 1562708406-95560d5755951e92fcc443295eb6685e.jpg

*	Current Date/Time + Encrypted filename
*	e.g: 201907092140-95560d5755951e92fcc443295eb6685e.jpg

*	Current Timestamp + Encrypted filename
*	e.g: 1562708406-95560d5755951e92fcc443295eb6685e.jpg

** Supported Image Extensions **

*	.jpg
*	.jpeg
*	.png
*	.gif

** Supported Files Extensions **

*	.pdf
*	.doc
*	.docx
*	.ppt
*	.pptx
*	.pps
*	.ppsx
*	.odt
*	.xls
*	.xlsx

** Supported Audio Extensions **

*	.mp3
*	.m4a
*	.ogg
*	.wav

** Supported Video Extensions **

*	.mp4
*	.m4v
*	.mov
*	.wmv
*	.avi
*	.mpg
*	.ogv
*	.3gp
*	.3g2
